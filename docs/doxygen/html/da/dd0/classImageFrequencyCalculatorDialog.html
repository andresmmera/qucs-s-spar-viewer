Detailed Description

A dialog for calculating the image frequency in superheterodyne receivers This calculator determines the image frequency based on the RF frequency, IF frequency, and local oscillator (LO) frequency. It supports both low-side and high-side injection modes.

The image frequency is calculated as:

  • Low-side injection: f_IM = f_RF - 2*f_IF
  • High-side injection: f_IM = f_RF + 2*f_IF
