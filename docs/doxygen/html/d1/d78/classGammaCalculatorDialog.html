Detailed Description

Dialog for calculating impedance, VSWR, and S11 from reflection coefficient (Γ)

This dialog provides a user interface for converting a complex reflection coefficient (Γ) into various impedance (Z), voltage standing wave ratio (VSWR), and S11 (dB).

Formulas:

  • Impedance: Z = Z₀ × (1 + Γ) / (1 - Γ)
  • VSWR: (1 + |Γ|) / (1 - |Γ|)
  • S11: 20 × log₁₀(|Γ|) dB
